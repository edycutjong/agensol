## Mission
Build identity-first applications on Solana using **SNS domains as the core identity primitive**. Two focus areas: **Agent Identity** (onchain AI agents with verifiable names) and **Social Identity** (human-readable profiles, social graphs).

## Track Scope & Focus Areas

### Agent Identity
- Agent registration with `.sol` domains
- Agent-to-agent messaging using SNS resolution
- Verifiable agent reputation tied to domain history
- Composable agent profiles

### Social Identity
- Domain-based profiles and social graphs
- Identity-gated access / token gates using `.sol`
- Cross-platform identity aggregation
- Human-readable payment addresses

## SDK / Integration Primitives
- `@bonfida/spl-name-service` — core SNS SDK
- Domain resolution / reverse lookup
- Record management (SOL, IPFS, URL, custom records)
- Sub-domain creation and management
- Domain transfer and tokenization
- Favourite domain system
